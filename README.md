## Project Description

This project focuses on developing a Reinforcement Learning based Dynamic Pricing System.

The objective is to build an intelligent pricing agent capable of adjusting prices dynamically based on inventory availability, customer demand, and remaining selling time.

The project will be implemented according to the roadmap provided by Infotact Solutions.
