# Infotact Internship Projects

Welcome to the **Infotact Technical Internship Program** workspace. This repository contains the codebases, research, and documentation for the advanced Data Science and Machine Learning (DS/ML) projects.

## Project Directory

| Project | Domain | Core ML Technique | Target KPI |
| :--- | :--- | :--- | :--- |
| **Project 1: Contextual Predictive Maintenance** | Manufacturing & Automotive | LightGBM + SMOTE + Contextual Data Fusion | Macro $F_1 \ge 0.85$ |
| **Project 2: Dynamic Pricing via RL** | Travel & Hospitality | Q-Learning / DQN (OpenAI Gymnasium) | Revenue > Heuristic Baselines |
| **Project 3: Geospatial Property Valuation** | Construction & Real Estate | GNN / Attention-Based Spatial Model | Minimum MAPE on Holdout Set |

## Getting Started
To get started with development, make sure you set up a virtual environment and install dependencies:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
pip install -r requirements.txt
```
